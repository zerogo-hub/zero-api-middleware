# zero-api-middleware

| 名称        | 作用                    |
| ----------- | ----------------------- |
| auth        | 基本认证，摘要认证      |
| bodylimit   | 限制请求体大小          |
| casbin      | 访问控制(未实现)        |
| cors        | 跨域控制                |
| jwt         | jwt 验证                |
| logger      | 请求日志                |
| must-param  | 必要参数检查            |
| newrelic    | 监控                    |
| nonce       | 随机参数 nonce 重复检查 |
| opentracing | 追踪                    |
| throttle    | 限流                    |
